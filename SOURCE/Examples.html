  Hint: Load example canvases using 'File/Load Canvas From Scene'!

Recreating the WebGL Demo

  1. Assign RTNodeEditor component to any GameObject - this is the example implementation of a runtime Node Editor
  2. (Optional) Setup the position on the screen, e.g. to fullscreen
  3. Load up the Node Editor Window and save some default canvases to the scene ('File/Save Canvas to Scene')
  4. Build (or play) and the runtime editor appears. You can load your presaved examples from the scene.

    5. Note: Technically, the runtime editor also supports saving/loading from/to XML-files, but you'll need a seperate file browser to load them - this is especially a challenge on WebGL
